📇 Contact Management System (Java)
📌 Overview

This project is a console-based Contact Management System developed in Java as part of my Software Development Internship (Task 03) at Prodigy InfoTech.
The application allows users to store, manage, and persist contact details such as name, phone number, and email.

🎯 Objective

To build an interactive user-driven application that demonstrates file handling, object serialization, and CRUD operations (Create, Read, Update, Delete) in Java.

🚀 Features
Add new contacts with Name, Phone, Email
View all saved contacts
Edit existing contacts
Delete contacts
💾 Stores data using file handling (contacts.dat)
🔄 Automatically loads saved contacts on restart
🛠️ Tech Stack
Language: Java
IDE: IntelliJ IDEA
Concepts Used:
Object-Oriented Programming (OOP)
Serialization (Serializable)
File Handling (ObjectInputStream, ObjectOutputStream)
ArrayList
Loops & Conditional Statements
▶️ How to Run
Open the project in IntelliJ IDEA
Compile the program
Run
Choose options from the menu to manage contacts
